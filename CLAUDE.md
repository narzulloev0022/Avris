# Avris AI — Контекст проекта для Claude

## Что это за проект

**Avris AI** — голосовая платформа медицинской документации для врачей Центральной Азии (основной рынок — Таджикистан и СНГ). Продукт создаётся компанией **Hyperion Labs**.

Основная идея: врач говорит голосом во время осмотра пациента, система автоматически транскрибирует речь и генерирует структурированную медицинскую документацию в формате SOAP.

---

## Текущее состояние (апрель 2026)

Репозиторий содержит один файл — `index.html` — полностью рабочий frontend-прототип (SPA без сборки). Бэкенда нет, все данные захардкожены в памяти.

## Экраны и функциональность

### Дашборд
- Hero-блок: кол-во активных пациентов, критических, средний Avris Score
- 3 метрики: SOAP за 24ч, сессии, точность AI
- Список пациентов с поиском и фильтрами (диагноз, отделение)
- Trend-chart Avris Score (bar, canvas)
- Timeline активности

### Осмотр (Consultation)
- Выбор пациента
- Кнопка микрофона с анимацией waveform и таймером
- Поле транскрипта (сейчас — typewriter-симуляция, нужен реальный Whisper)
- SOAP-форма: S / O / A / P (4 textarea)
- Кнопка «Подтвердить SOAP» → запись в историю

### Ночной обход
- Сетка палат (6 штук, статус stable/warning)
- Выбор палаты → голосовой ввод (сейчас захардкожен, нужен Whisper)
- Поля: Осмотр + План лечения
- Сохранение → запись в историю

### История
- Список записей SOAP и обходов (in-memory, сбрасывается при перезагрузке)

### Реанимация (ICU Monitor)
- ОРИТ: 3 пациента с витальными показателями (ЧСС, АД, SpO2, ЧДД, T°C, дни)
- Пост-реанимация: 2 пациента
- Анимированные статус-точки (stable/warning/critical)
- Алертные состояния: красный при критических показателях

### Настройки
- Тема (светлая/тёмная)
- Язык интерфейса
- Информация об аккаунте
- Выход
