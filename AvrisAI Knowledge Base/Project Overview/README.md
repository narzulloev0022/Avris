# Обзор проекта

## Что такое Avris AI

**Avris AI** — первая платформа голосовой медицинской документации для Центральной Азии. Врач говорит голосом во время осмотра или ночного обхода, система автоматически распознаёт речь и генерирует структурированную документацию в формате SOAP.

Создаётся компанией **Hyperion Labs**.

---

## Бизнес-модель

### Проблема
Врачи в Таджикистане и СНГ тратят 30–40% рабочего времени на бумажную документацию. EMR-системы неудобны, часто на русском/английском, без голосового ввода.

### Решение
- Голосовой ввод на родном языке (русский, таджикский, узбекский)
- Автоматическая SOAP-документация через AI
- Интеграция с ночными обходами и мониторингом ОРИТ

### Целевые сегменты
- **ICP:** Врачи-терапевты, пульмонологи, кардиологи в государственных больницах Таджикистана
- **Расширение:** Частные клиники СНГ, Казахстан, Узбекистан

## Конкурентная среда

| Конкурент | Рынок | Слабость |
|-----------|-------|---------|
| Nuance DAX | США/Европа | Нет локализации под СА, дорого |
| Suki AI | США | Только английский |
| DeepScribe | США | Нет СА-рынка |
| Локальные EMR | Таджикистан/СНГ | Нет голоса, плохой UX |

**Преимущество Avris:** Локальный язык + контекст медицины СНГ + цена.
