# Продукт

## Текущая версия: v3.5

Приложение представляет собой SPA — один файл `index.html`, запускается без сборки прямо в браузере. UI готов к демонстрации, бэкенд предстоит реализовать (Whisper + Claude Sonnet).

---

## Клинический workflow — 4 этапа приёма

Полный цикл работы врача с одним пациентом:

### 1. Подготовка (Pre-encounter)
- Открытие пациента из списка → загружается **карточка пациента** с Avris Score, текущими диагнозами, аллергиями, назначениями, анамнезом
- Просмотр витальных показателей за 7 дней (АД / ЧСС / T°C / SpO₂)
- Insight-фраза от AI: контекст для врача («Ночное давление >150, усилить контроль»)

### 2. Осмотр (Encounter / Consultation)
- Врач переходит на экран Consultation — pat-context уже подгружен
- Нажимает кнопку микрофона → Whisper транскрибирует речь в реальном времени
- Claude Sonnet анализирует транскрипт → автозаполняет поля **S / O / A / P**
- Evidence Link: кликнул фразу в транскрипте → подсветилась карточка S/O/A/P, к которой она относится
- Врач вычитывает, корректирует → «Подтвердить SOAP» → запись попадает в историю

### 3. Анализы (Lab Connect)
- Кнопка «Направить на анализ» в pat-context → модалка с **6 группами** (~38 тестов)
- Поиск, групповой select-all, счётчик выбранных
- Генерируется QR-направление → пациент идёт в лабораторию
- Лаборатория сканирует QR → загружает PDF/данные через портал
- Результаты появляются на вкладке «Анализы» в Consultation
- Claude Sonnet добавляет **AI-комментарий** к таблице показателей

### 4. Ночной обход (Night Round)
- Сетка 6 палат со статусом (stable/warning/critical)
- Врач выбирает палату → modal с двумя полями (Осмотр + План лечения)
- FAB-кнопка с микрофоном для голосового авто-заполнения
- Сохранение → запись в `histData` → отображается в Activity Timeline на дашборде

В фоне: ICU Monitor показывает критических пациентов в реальном времени, доктор может среагировать и вызвать коллегу прямо из карточки.

## Агент 1 — Осмотр пациента (Consultation)

**Назначение:** Запись голосового осмотра и автоматическая генерация SOAP-документации.

### Текущий статус
- ✅ UI полностью готов (3 вкладки: Осмотр / Анализы / История)
- ✅ Анимация waveform, таймер записи, REC-индикатор
- ✅ SOAP-форма с 4 полями (S/O/A/P) + AI-бейдж Claude Sonnet
- ✅ STT lang switcher (RU/TJ/EN, независим от UI)
- ✅ Evidence Link — кликабельные фразы в транскрипте подсвечивают SOAP
- ❌ Реальная голосовая запись (сейчас typewriter-симуляция)
- ❌ Whisper транскрипция через API
- ❌ Claude Sonnet → автогенерация SOAP

### Поля SOAP
- **S (Subjective)** — жалобы пациента

- **P (Plan)** — план лечения

## Lab Connect — Killer Feature

**Полный спек:** см. [[LAB-CONNECT]]

### Концепция
Интеграция лабораторных результатов через QR-направления прямо в карточку пациента. От направления до AI-комментария — без выхода из приложения.

### Workflow
1. Врач нажимает «Направить на анализ» в pat-context
2. Модалка: поиск, 6 групп тестов, счётчик выбранных, кнопка «Выбрать все в группе»
3. После «Распечатать направление» генерируется уникальный QR-код
4. Пациент показывает QR в лаборатории
5. Лаборант сканирует → открывается портал загрузки
6. Лаборант загружает PDF результатов или вводит данные
7. Врач получает уведомление → результаты на вкладке «Анализы»
8. **Claude Sonnet** автоматически анализирует результаты и добавляет комментарий

### Группы тестов (38 анализов)
| Группа | Кол-во | Примеры |
|--------|--------|---------|
| **Clinical (Клинические)** | 4 | ОАК / ОАМ / Группа крови и резус / Коагулограмма |
| **Biochem (Биохимия)** | 8 | Глюкоза, общий белок, АЛТ/АСТ, креатинин/мочевина, холестерин, билирубин, железо/ферритин, HbA1c |
| **Vitamins (Витамины)** | 4 | D, B12, фолиевая кислота, Mg/K/Ca |
| **Infectious (Инфекции)** | 7 | ВИЧ, гепатит B/C, малярия, сифилис, COVID-19, туберкулёз |
| **Hormones (Гормоны)** | 6 | ТТГ, T3/T4, кортизол, тестостерон/эстрадиол, ПСА, ХГЧ |
| **Instrumental (Инструментальные)** | 9 | ЭКГ, рентген ГК, УЗИ ОБП, ЭхоКГ, МРТ мозга, КТ ГК, ФГДС, спирометрия, Холтер |

### UX-находки
- Поиск работает по локализованному названию теста — врач может вбить «hba1c» в EN и найдёт «гликированный гемоглобин» в RU-версии (если переключит)
- «Выбрать все в группе» работает **только с видимыми** строками: если поиск отфильтровал часть, скрытые не выберутся
- Состояние выбора (`labChecked`) переживает смену языка, что критично для двуязычных команд

### Бизнес-ценность
- Врач не сходя с места получает все данные пациента
- Время на документацию сокращается на 80%
- Лаборатории получают цифровой workflow — сетевой эффект
- Основа для государственной электронной медкарты (СНГ-перспектива)

## Avris Score

**Что это:** Числовой показатель (0–100) состояния пациента. Агрегируется из витальных показателей и AI-оценки.

| Диапазон | Статус | Цвет | TR-ключ статуса |
|----------|--------|------|-----------------|
| 0–39 | Критический | `#EF4444` | `st_crit` |
| 40–59 | Наблюдение | `#F59E0B` | `st_watch` |
| 60–79 | Умеренный | `#3B82F6` | (используется только цвет) |
| 80–100 | Стабильный | `#10B981` | `st_stab` |

Отображается в badge списка пациентов, в pat-context, в SVG-ring карточки пациента, на дашборде в trend-bars.

## ICU Monitor (Реанимация)

### Витальные показатели и алерты
- ЧСС (уд/мин) — алерт при `>100`, warning при `>90`
- АД (мм рт.ст.) — алерт при `<90` (систолическое)
- SpO₂ — алерт при `<90%`, warning при `<94%`
- ЧДД (вд/мин), Температура (°C), Дни в ОРИТ

### Структура bed
Кровати теперь в форме `bedT: "icu" | "post"` + `bedN: number`. Префиксы локализованы:
- RU: «ОРИТ» / «ПР»
- TJ: «ОРИТ» / «ПП»
- EN: «ICU» / «RW»

Кнопка «Вызвать врача» (с SVG-иконкой молнии) появляется только для critical/warning пациентов.

## Платформа — общие фичи

### Мультиязычность
Полная локализация: 🇷🇺 Русский (default) / 🇹🇯 Тоҷикӣ / 🇬🇧 English. Все 6 пациентов имеют EN-переводы диагнозов, анамнеза, назначений.

### Тёмная / светлая тема
- Светлая: RAL 9016 Pure White (`#F1F0EA`)
- Тёмная: Deep Navy
- Переключатель в topbar и на login-экране, persist через localStorage

### Lab AI Insight
Статичный пример демонстрирует возможности: «Лейкоцитоз и повышенное СОЭ указывают на воспалительный процесс. Анемия лёгкой степени. Рекомендую дополнить биохимией крови.» — будет генерироваться Claude Sonnet'ом в продакшене.

### Карточка пациента
- Полный профиль: возраст, пол, рост, вес (через `t("unit_cm/kg")`), BMI, группа крови (RU/EN нотация)
- Текущие диагнозы, анамнез, аллергии, назначения — все с EN-вариантами
- Avris Score ring + insight-фраза
- Витальные показатели — sparkline за 7 дней, 3 ключевые метрики (ЧСС/АД/SpO₂)

### Time Saved Banner
На дашборде: «Время сэкономлено сегодня: 1.4 часа». Будет считаться по факту в продакшене как разница между средним временем оформления SOAP вручную (~12 мин) и через Avris (~2 мин).
