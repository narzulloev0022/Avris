# Продукт

## Текущая версия: v3.5

Приложение представляет собой SPA (Single Page Application) — один файл `index.html`, запускается без сборки прямо в браузере.

---

## Агент 1 — Осмотр пациента (Consultation)

**Назначение:** Запись голосового осмотра и автоматическая генерация SOAP-документации.

### Текущий статус
- ✅ UI полностью готов
- ✅ Анимация waveform, таймер записи
- ✅ SOAP-форма (4 поля: S/O/A/P)
- ❌ Реальная голосовая запись (сейчас симуляция)
- ❌ Whisper транскрипция
- ❌ Claude Sonnet → генерация SOAP из транскрипта

### Флоу (целевой)
```
Врач нажимает 🎙️
  → Whisper API транскрибирует речь
  → Транскрипт отображается в реальном времени
  → Claude Sonnet анализирует транскрипт
  → Автозаполнение полей S / O / A / P
  → Врач редактирует → «Подтвердить SOAP»
  → Запись сохраняется в историю
```

### Поля SOAP
- **S (Subjective)** — жалобы пациента
- **O (Objective)** — данные осмотра
- **A (Assessment)** — диагноз
- **P (Plan)** — план лечения

## Avris Score

**Что это:** Числовой показатель (0–100) состояния пациента. Агрегируется из витальных показателей.

| Диапазон | Статус | Цвет |
|----------|--------|------|
| 0–39 | Критический | 🔴 Красный |
| 40–69 | Наблюдение | 🟡 Жёлтый |
| 70–100 | Стабильный | 🟢 Зелёный |

Отображается:
- В списке пациентов (badge)
- В модальном окне пациента (SVG ring-chart)
- На дашборде (trend bars)

## ICU Monitor (Реанимация)

**Назначение:** Визуальный мониторинг пациентов в ОРИТ и пост-реанимации.

### Витальные показатели
- ЧСС (уд/мин) — алерт при >100
- АД (мм рт.ст.) — алерт при сист. <90
- SpO₂ — алерт при <90%, warning при <94%
- ЧДД (вд/мин)
- Температура (°C)
- Дни в ОРИТ

### Статусы пациентов
- `stable` — зелёная точка
- `warning` — жёлтая точка, пульсирует
- `critical` — красная точка, быстро пульсирует

## Фичи платформы

### Мультиязычность
Полная локализация на 3 языка:
- 🇷🇺 Русский (default)
- 🇹🇯 Тоҷикӣ (таджикский)
- 🇬🇧 English

### Тёмная / светлая тема
Полностью реализовано через CSS-переменные. Переключатель в topbar и на экране логина.

### Модальное окно пациента
- Полный профиль: возраст, пол, рост, вес, BMI, группа крови
- Текущие диагнозы, анамнез, аллергии
- Назначения (медикаменты)
- Avris Score ring + insight-фраза
- Витальные показатели — canvas-чарт с 4 табами (АД/ЧСС/T°C/SpO₂) за 7 дней

### Поиск и фильтрация пациентов
- Поиск по ФИО и диагнозу (debounced 200ms)
- Фильтр по диагнозу
- Фильтр по отделению
- Кнопка «Сбросить»
