# Материалы для инвесторов

## Нарратив питча

### Проблема
Врачи в Таджикистане и странах СНГ тратят до 40% рабочего времени на ручное заполнение медицинской документации. Существующие EMR-системы:
- Неудобны, требуют ввода с клавиатуры
- Не имеют голосового ввода
- Не адаптированы под местные языки (русский, таджикский)
- Дорогие западные аналоги не локализованы

### Решение — Avris AI
Первая голосовая AI-платформа для медицинской документации в Центральной Азии:
- Врач говорит → система автоматически создаёт SOAP-документацию
- Поддержка русского, таджикского, узбекского языков
- Работает на телефоне в условиях больницы
- Интеграция с ночными обходами и мониторингом ОРИТ

### Почему сейчас
- OpenAI Whisper достиг достаточного качества для медицинской речи
- LLM (Claude Sonnet) позволяет структурировать неструктурированную речь в клинические форматы
- Рынок Центральной Азии не охвачен ни одним игроком

### Почему мы
- Глубокое понимание рынка Таджикистана/СНГ
- Локальные языки в основе, не как дополнение
- v3.5 прототип с полным UI уже готов

**Q: Кто конкуренты?**
A: В СНГ — никого. Глобально: Nuance DAX, Suki AI, DeepScribe — все работают только в США/Европе, без поддержки русского/таджикского, стоимость $500–1000/врач/месяц. Avris входит с локализацией и ценой для рынка СНГ.

**Q: Почему не сделают конкуренты?**
A: Nuance/Microsoft сфокусированы на рынке США ($50B+). Локальный рынок СНГ требует языковой экспертизы, понимания советской медицины (SOAP-нотация, МКБ-10), локальных регуляторных требований.
